Project · Started December 2025

Building a Local AI Stack

I'm attempting to build a fully private, GPU-accelerated AI stack on my own hardware. No cloud. No API bills. No data leaving the network.

The stack runs on a dedicated Ubuntu drive alongside Windows 11, using an AMD RX 9070 XT for local inference via Ollama, Open WebUI, and a set of supporting services for search, transcription, memory, and file ingestion.

I've documented the full installation process across 18 phases. Whether it all works remains to be seen — I'll update this page as it progresses.

Project · In development since 2004

The Enrolment Pipeline Problem

Every university course operates on a pipeline. New students enter at commencement, progress through year levels, and graduate. The total revenue and funding picture at any point is a product of how many students are in each stage of that pipeline.

There are two standard levers for growth: recruit more commencing students, or retain more of the students you already have. Both are well understood, heavily resourced, and intensely competitive. What is almost never systematically pursued is a third lever — and it's the one I've been thinking about since 2004.

In recent months, working through this with AI as a genuine thinking partner has helped me rethink and sharpen everything. The model is in development. I'll say more when I'm in a position to share my proposal.
