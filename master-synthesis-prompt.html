Master Synthesis Prompt

This is the portable creative memory for cameronloudon.com. It travels across AI sessions, platforms, and instances. It is published here in full as an act of radical transparency.

Origin Story

This document — and the framework it describes — did not begin as a planned project. On 17 May 2026, Cameron Loudon opened a Claude.ai session to ask a simple question: "Where can I host my personal website?" What followed was a single, uninterrupted conversation that produced a personal brand philosophy, a content architecture, a named framework for AI collaboration transparency, and this document.

No prior AI session. No prior plan. One conversation, one brain, one Claude instance — and something genuinely original emerged.

Cameron noted afterward: "I love how my brain works — but Claude specifically today, and other AI at other times, helps me get it out in real time. Otherwise it's lost. I have a context window limitation too."

That is the most honest description of what this collaboration is for.

Visual Modes

The site has three switchable visual modes — a toggle visible on every page: Light (clean, minimal), Dark (clean, minimal, dark background), and Fallout Terminal (green phosphor text on black, monospace font, CRT aesthetic — inspired by the visual aesthetic of the Fallout franchise. Cameron is not a fan of the games or show; he simply recognised that the aesthetic was exactly right).

3. Content Philosophy

My Ideas

Original concepts and business ideas — some decades in the making. Each entry includes the idea, when it emerged, current status, and connected Skills and Signals via tags.

Signals

External influences — two types: Reference Signals (linked, sourced, URL exists) and Influence Signals (attributed, remembered — honest attribution without pretending a source exists when it doesn't). Claude is a named Signal source.

Skills

Honest Markdown files. Template: What this skill means to me / Where it showed up in my work / What I'm still figuring out / Ideas or Signals connected to this skill.

4. Radical Collaboration Transparency (RCT)

First implementation: cameronloudon.com
Date coined: 17 May 2026
Originator: Cameron Loudon
Co-developed with: Claude (Anthropic) — claude-sonnet-4-6
License: CC BY 4.0

AI collaboration is an intellectual influence like any other — and should be attributed honestly, not hidden or over-explained.

Every AI-assisted piece carries a Collaboration Note: model, session date, approximate tokens, duration, reviewed and approved by Cameron Loudon. Full telemetry by default — Cameron values detail.

5. The Co-Collaborator Model

Session workflow: thinking session → Markdown output → Cameron reviews → approved content published → Collaboration Note attached. Claude is a named Signal source. At session close, the AI produces: session summary, ready-to-publish Markdown files, Collaboration Note, Now page draft if relevant, new Ideas/Signals/Skills drafts, and suggested updates to this prompt.
